# LoadLink India 🚚 🇮🇳

**Smart Goods Transport Platform - Simplified for Everyone.**

LoadLink India is a professional, mobile-first web application designed to connect van drivers and load owners across India. Inspired by platforms like Porter, it simplifies the logistics process by providing instant pricing and a unique negotiation system.

---

## ✨ Features

- **🏠 Home View**: Instant role selection for Customers and Drivers.
- **📦 Customer Flow**: 
  - Post loads with pickup/drop locations and goods type.
  - Choose between Tata Ace, Pickup, or Bike.
  - Instant pricing based on a distance-rate formula.
  - Suggest your own price for negotiation.
- **🚛 Driver Flow**:
  - Browse available loads in real-time.
  - Accept jobs instantly or counter-offer with a new price.
  - Integration with Call and WhatsApp for immediate coordination.
- **💰 Commission System**: Automatic 10% platform fee calculation.
- **🗄️ Persistence**: Saves all data locally using `localStorage` (No backend required).
- **🎨 Premium UI**: Clean, responsive, and animated using Vanilla HTML/CSS/JS.

---

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+).
- **Icons**: Font Awesome 6.
- **Typography**: Google Fonts (Outfit).
- **Data Persistence**: ブラウザの `localStorage`.

---

## 🚀 Getting Started

1. Clone or download this repository.
2. Open `index.html` in any web browser.
3. Choose your role and start linking loads across India!
